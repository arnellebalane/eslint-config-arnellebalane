# eslint-config-arnellebalane

My personal ESLint configuration.


## Installation

```bash
$ yarn add --dev eslint-config-arnellebalane  // or:
$ npm install --save-dev eslint-config-arnellebalane
```


## Usage

```js
// .eslintrc
{
    "extends": "arnellebalane",
    "rules": {
        // your overrides for my rules
    }
}
```
